# 💕 Our Little Website

Static site — deploy to GitHub Pages in 5 minutes.

## Files

```
index.html              ← password page
home.html               ← home page
static/
  css/style.css
  js/petals.js          ← floating hearts/petals animation
  js/login.js           ← password check
  js/home.js            ← days-together counter + scroll reveals
```

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "💕 first commit"
git remote add origin https://github.com/YOU/REPO.git
git push -u origin main
```

Then: **Settings → Pages → Branch: main / root → Save**

Your URL: `https://YOUR_USERNAME.github.io/YOUR_REPO/`
